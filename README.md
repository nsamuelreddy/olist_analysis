# Olist E-commerce Data Analysis

##  Project Overview
This project analyzes the Olist e-commerce dataset using SQL to extract business insights such as revenue, order trends, and top-selling products.

##  Tools Used
- MySQL
- SQL
- MySQL Workbench

##  Key Insights
- Total number of orders
- Revenue generated from all transactions
- Top 5 best-selling products
- Monthly order trends
- Customer-wise revenue analysis

##  Project Structure
- `queries/analysis.sql` → All SQL queries
- `outputs/` → Query results & screenshots

##  What I Learned
- Writing complex SQL queries
- Aggregations and joins
- Business insight extraction from raw data

##  Dataset
Olist Brazilian E-commerce Dataset (Kaggle)
